# AI Consulting Project

Проект состоит из двух частей:
- **/kaz-invest**: Фронтенд на Vue 3 (Vite).
- **/serverInvest**: Бэкенд на Node.js (Express) с интеграцией OpenRouter.

## Как запустить
1. Склонируйте репозиторий.
2. В папке `serverInvest/server` создайте `.env` с ключом `AI_API_KEY`.
3. Запустите бэкенд: `cd serverInvest/server && npm install && node server.js`
4. Запустите фронтенд: `cd kaz-invest && npm install && npm run dev`
